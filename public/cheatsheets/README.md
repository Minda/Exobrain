# Cheatsheets

HTML-based reference pages for various tools, technologies, and concepts.

## Index

Open **[index.html](index.html)** for a categorized overview of all cheatsheets.

## Contents

- **exo-frontier-ai.html** - Reference for exo, a distributed AI inference system

## Usage

Open any HTML file directly in your browser to view the cheatsheet.

## Structure

Each cheatsheet is a self-contained HTML file with:
- Inline CSS for styling
- No external dependencies (except Google Fonts)
- Mobile-responsive design
- Dark theme optimized for readability

## Adding New Cheatsheets

Simply add new HTML files to this folder. Each cheatsheet should be:
- Self-contained (single HTML file)
- Well-structured with clear sections
- Include inline styling
- Named descriptively (e.g., `tool-name.html` or `concept-overview.html`)
